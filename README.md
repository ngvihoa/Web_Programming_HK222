# CV_Management

Web Programming project -  CV management web-base application.

Please follow these steps to run our group web app on your computer:

### Step 1: Download and install XAMPP

### Step 2: Download source code from github to xampp/htdocs folder

The code must be stored here to run, other directories are not allowed.
The folder name should be CVblog after extracted.

### Step 3: Open XAMPP control panel and turn on Apache and mySQL

### Step 4: Click "Admin" button of mySQL to go to its dashboard

### Step 5: Create a database and name it "cvwebsite"

### Step 6: Import database.sql file to create the database

### Step 7: Change config.php
If you want to run this web app follow your computer configurations, you can go to this file and change DBNAME, DBHOST, DBUSER, DBPASS, and DBDRIVER.

### Step 8: Open your web browser and type http://localhost/Cvblog/public in the url bar to start the quiz app

### Step 9: Log in to start using

Try **username: HoaNguyen** and **password: 12345678** to log in.

You can also create new account to log in.

**Note:** 
Make sure that .htaccess and robots.txt always exist.

## **If you have correctly followed the above steps then the web app should run perfectly fine on your computer.**
